## Findings considered and rejected (do not re-audit)

Surfaced during the 2026-06-18 audit at commit `8f09a1d` but explicitly
**not** turned into plans this round:

- **Drop the legacy `transactions.tradingLogic` column**: writes still
  exist (`server/routers.ts:155, 469`); dropping requires a migration
  plus rewriting many `*.lifecycle.test.ts` SQL fixtures. Value-to-cost
  at single-user local scale is low; revisit if the column starts
  blocking a real change.
- **Upgrade Express 4.22.x ‚Üí 5.x to drop the moderate `qs` advisory**:
  breaking change in Express's error/router model; single-tenant
  localhost-only server makes the DoS surface notional.
- **Validate `tvUrl` / `reviewChartUrl` to reject `javascript:` URLs**
  (`client/src/pages/Transactions.tsx:511`,
  `client/src/pages/TransactionDetail.tsx:328,432`): the trader pastes
  their own URLs into their own journal; there is no other user to
  attack. Single-tenant local app.
- **Component-LOC violations in `TransactionDetail.tsx` (463),
  `Accounts.tsx` (425), `DashboardLayout.tsx` (228)**: also over the
  200 LOC rule, but lower ROI than 010/011/012. Carry to a future batch
  if 010/011/012 land cleanly.
- **`vite-plugin-manus-runtime` + `client/public/__manus__/debug-collector.js`**:
  Manus sandbox debug instrumentation; removal is a larger decision
  (does this project still ship through Manus?). Plan 006 explicitly
  keeps both in scope.
- **`next-themes` `useTheme()` call in `client/src/components/ui/sonner.tsx`
  with no `NextThemeProvider` in the app tree**: returns sensible
  defaults; works correctly. Cosmetic.
- **Sidebar nav labels "Dashboard"/"Transactions"/‚Ä¶ capitalised vs.
  `DESIGN.md`'s lowercase-label rule**: re-deferred. Chrome rather
  than data; the author already overrode this once.

Carry-forward rejections from the 153587a batch (still rejected):

- **`transactions.list` returns `SELECT *`**: single-user scale;
  composite indexes already cover the hot queries.
- **`transaction.update` zod input accepts close/review fields the
  router then rejects**: router enforces the FSM; harmless duplication.
- **`pnpm.patchedDependencies` references `wouter@3.7.1` while deps
  pin `^3.3.5`**: repo uses npm; pnpm metadata is inert.
- **Capitalised "Trading Journal" wordmark in `DashboardLayout`**:
  same chrome exception.
